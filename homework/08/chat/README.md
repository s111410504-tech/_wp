# 專案名稱-「劇」在一起
>本專案使用 Opencode MiniMax M2.5 輔助生成

## 專案介紹

### 1.動機
本身很喜歡追劇跟看電影，所以想做一個社群，上面都是可以互相推薦好看的劇跟電影，也可以分享心得。
### 2.目標
用戶可以在社群上分享心得，遇到相同愛好的人可以私訊聊天，同時也可以透過社群知道最近上映的電影跟戲劇。
### 3.功能
**登入/註冊**
- 使用者名稱和密碼登入
- 新用戶可註冊帳號
- 刷新頁面保持登入狀態（每個瀏覽器標籤頁獨立）  

**分享牆**
- 查看所有用戶發布的貼文
- 按愛心、按留言
- 點擊頭像進入該用戶主頁    

**個人主頁**
- 發布文字貼文
- 上傳照片
- 刪除自己的貼文
- 編輯個人資料（暱稱、簡介、喜歡的類型）  

**他人主頁**
- 查看該用戶所有貼文
- 按愛心、留言
- 按「私訊」直接對話  

**側邊欄**
- 近期熱映電影/劇推薦

## 3.心得
我這個專案大多是用opencode做出來的，雖然是用AI但是我覺得還是要一些技巧，要怎麼明確地說出現在的需求很重要，所以我後來大多是用條列式說明我的需求，大概的方向做出來之後再慢慢去調整。我認為看懂程式碼也很重要，因為專案的側邊頁是會條列出現在熱映的電影跟戲劇，這部分我是自己手動調整，必須要找出這段程式碼寫在那再去進行修改。
