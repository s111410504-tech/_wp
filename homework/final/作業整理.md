- **檔案**：10 個 JS 檔案 + README.md
- **AI 對話連結**：https://gemini.google.com/share/c2f34c52ed01

### 專案概述
包含 10 道 JavaScript 進階練習題，主要聚焦於 **Closure（閉包）** 和 **Higher-Order Function（高階函式）**。

### 核心概念

#### Higher-Order Function（高階函式）
- 將函式作為參數傳入
- 回傳函式作為結果

#### Closure（閉包）
- 函式可以記住其詞法作用域
- 即使外部函式已執行完畢，仍可存取其變數

### 代表性程式碼範例

```javascript
// 高階函式 (mathTool)
function mathTool(num1, num2, action){
    return action(num1, num2);
}

const plus = mathTool(10, 5, function(a, b){return a+b;});
const minus = mathTool(10, 5, function(a, b){return a-b;});

// 閉包 (multiplier)
function multiplier(factor) {
    return function(number) {
        return number * factor;
    };
}

const double = multiplier(2);
console.log(double(5));  // 10
```

### 學習收穫
- 深入理解高階函式的概念與應用
- 掌握 Closure 的運作原理
- 學習使用 map、filter、reduce 等陣列方法

包含 10 道 Node.js 進階練習題，主要聚焦於 **物件操作**、**Callback Pattern（回呼模式）** 和 **非同步處理**。

### 核心概念

#### 物件操作
- 點符號與中括號存取
- 物件屬性的增刪改查

#### Callback Pattern（回呼模式）
- 函式作為參數傳遞
- 錯誤優先回呼

#### 非同步處理
- 模擬非同步資料取得
- 錯誤處理與驗證

### 代表性程式碼範例

```javascript
// 物件宣告與存取
const post = {
    id: 1,
    title: "Hello World",
    content: "Markdown content"
};

console.log(post.title);      // 點符號
console.log(post["title"]);   // 中括號

// Callback 模式
function fetchData(callback) {
    setTimeout(() => {
        const data = { id: 1, name: "測試資料" };
        callback(null, data);
    }, 1000);
}
```

### 學習收穫
- 掌握 JavaScript 物件操作技巧
- 理解 Callback Pattern 的運作原理
- 學習非同步程式的錯誤處理方式

## 08. 社群軟體專案「劇」在一起

### 基本資訊
- **作業類型**：全端社群平台專案
- **技術棧**：Node.js + Express + SQLite3 + HTML/CSS/JavaScript
- **檔案結構**：`chat/` 目錄
- **輔助工具**：使用 Opencode MiniMax M2.5 輔助生成

### 專案概述
以影視為主題的社群平台，讓使用者可以分享追劇心得、推薦好劇，並與同好交流。

### 動機與目標
- **動機**：本身喜歡追劇和看電影，想做一個可以互相推薦劇和電影的社群
- **目標**：
  - 使用者可以在社群上分享心得
  - 遇到相同愛好的人可以私訊聊天
  - 透過社群知道最近上映的電影跟戲劇

### 功能模組

#### 1. 登入/註冊系統
- 使用者名稱和密碼登入
- 新用戶可註冊帳號
- 刷新頁面保持登入狀態

#### 2. 分享牆
- 查看所有用戶發布的貼文
- 按愛心、按留言
- 點擊頭像進入該用戶主頁

#### 3. 個人主頁
- 發布文字貼文
- 上傳照片
- 刪除自己的貼文
- 編輯個人資料（暱稱、簡介、喜歡的類型）

#### 4. 他人主頁
- 查看該用戶所有貼文
- 按愛心、留言
- 按「私訊」直接對話

#### 5. 側邊欄
- 近期熱映電影/劇推薦

### 技術細節
- **後端**：Express.js + sqlite3（原生 SQLite3）
- **前端**：單一 `index.html` 檔案（SPA 架構）
- **特色功能**：即時打字狀態、照片上傳、私訊聊天

### 資料表設計
```sql
-- 使用者表
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    password TEXT NOT NULL
)

-- 貼文表
CREATE TABLE posts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    content TEXT NOT NULL,
    image_path TEXT
)

-- 私訊表
CREATE TABLE messages (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    sender_id INTEGER NOT NULL,
    receiver_id INTEGER NOT NULL,
    content TEXT NOT NULL
)
```
